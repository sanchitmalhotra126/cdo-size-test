Local School Database API

We've enabled JSON access to the database that powers the Code.org local school search. Our goal is to enable third-parties to take advantage of our database, in order to build local search engines or other applications beyond the simple local search we provide. All information that is publicly available via our search engine is exposed via JSON - public contact info is listed, but private contact info is not.

Get the Code.org local schools database (JSON)

To learn more about the JSON format and how to integrate it with your own custom apps, please visit http://www.json.org.

The data behind the Code.org school database is licensed via the Creative Commons attribution license - you can use the data as you wish, but please give Code.org attribution.
